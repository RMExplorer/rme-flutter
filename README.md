# RME Mobile

Reference Material Explorer Mobile
The RM Explorer Mobile is a native mobile application that enables data visualization and analysis based on the NRC digital repository of reference material certificates
and integrates it with other data sources such as PubChem and the BIPM Key Comparison Database. 

The RM Explorer was developed by the National Research Council of Canada's (NRC) Biotoxin Metrology and Inorganic Metrology Teams.
